Инструкция: https://github.com/gulpjs/gulp/blob/master/docs/getting-started.md

при первом запуске: 
1)устоновить nodejs:					https://nodejs.org/en/
2)в Командной Строке(КС) прописать: 	npm install --global gulp-cli


Работа с проектом:
	все действия происходят в папке куда хотим поместить gulp,
	что бы вызвать КС с адресом папки зажимаем Shift+правая кнопка мыши в выповшем меню "Открыть окно каманд"
Устанавливаем Gulp в проект:
1)	копируем файл:						package.json
3)	копируем файл:						gulpfile.js
для загрузки необходимых файлов использовать команду в КС :  npm i
Настройка Gulp:
	Вся настройка происходит в файле gulpfile.js после установки плагинов. Плагины искать здесь:  https://gulpjs.com/plugins/

	Команда в КС для локальной установки плагина: npm install --save-dev (Имя_плагина)


Необходимо сохранять структуру папок
папка 	/app/ - рабочая папка
		/app/sass - кидаем наши стили сюда
			/sass/mystyle.sass - сюда подключаем все стили 
			/sass/libs.sass - сюда подключаем стили из старонних библиотек.
			
		/app/libs - кидаем староннии библиотеки сюда
			js файлы библиотек подключаются в gulpfile.js в gulp.task('scripts')

Выгрузка готового проекта осуществляется по команде в КС:  	gulp build
		все файлы выгрузятся в папку
		/dist/

Установка старонних Библиотек используем :  https://bower.io/ также требует установленого Git
	.bowerrc - указывает путь куда будут сохранятся библиотеки
	для загрузки в КС прописать: bower i имя_библиотеки
